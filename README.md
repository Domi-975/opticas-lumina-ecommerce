
# Ópticas Lumina - E-commerce

Proyecto colaborativo de e-commerce para Ópticas Lumina. Full-stack con React (frontend) y Express (backend).

## Setup
1. Clona el repo: `git clone https://github.com/tu-usuario/opticas-lumina-ecommerce.git`
2. Instala dependencias:
   - Frontend: `cd frontend && npm install`
   - Backend: `cd backend && npm install`
3. Ejecuta:
   - Frontend: `cd frontend && npm run dev` (http://localhost:5173)
   - Backend: `cd backend && npm run dev` (http://localhost:5000)

## Estructura del proyecto
- `frontend/`: React app con Vite.
- `backend/`: API con Express y PostgreSQL.
